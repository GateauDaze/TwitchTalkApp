Basic Usage

  1. At first, launch "Bouyomi-chan".
  2. Click [Connect with Twitch] and login to Twitch.
  3. Enter your username to "Username" box.
    * username means your login ID.
  4. Enter channel ID that you want to read.
    * If you want to read own channel, enter your own ID.
    * If you want to read other channel, enter the specific ID.
    * You can set multiple channels with comma delimited.
  5. After pressing the "Connect" button, voice robot will talk chat messages.

Export Debug Log

  1. When you set "-log" launch option of TwitchTalkApp.exe, logfile will be export.
  2. You can easily set launch option in command prompt or create shortcut link.

Settings

Replacement

Before reading, you can replace specific words in message.
Replacement settings is separated between Japanese and English.
If you replace English word to Japanese word, SpeechAPI won't read it.
You can set Emoticon Replacement separately from previous replacement.

Emotes will be replaced before Japanese/English replacement.
Input specific pronunciation into "Pronunciation", and press Apply button.
When replaced emote is Japanese, it will be read by Bouyomi-chan.
Japanese/English replacement will be also applied to replaced emotes.

JP Voice Settings

You can change connection settings to "Bouyomi-chan".
If your "Bouyomi-chan" socket port is not 50001, you can change in here.
If you launch "Bouyomi-chan" in other PC, you can change host address in here.

EN Voice Settings

You can change English voice volume, speed and pitch.
If you un-checked "Use English Voice", English message will be read by Bouyomi-chan.
This options may be no effect on some PC.

Edit Blocked Users

You can edit Blocked Users. Choosed user' comment won't be read.
Nightbot is set in default.
When comment posted, the user will listed up automatically.
If you toggle to "Unread", comments which posted by specific user won't be read.

Options

Read Username

If you check this, username who posts comment will be read after message.
Username also be replaced.

Read Emoticons

If you check this, emotes in the message will be read.

Use English Voice

If you check this, English message will be read by SAPI5 voice which set in EN Voice Settings.
When this is unchecked, English message will be read by same voice as Japanese message.

Use Notification

If you check this, notificate comment with popup.
When you play a game with fullscreen, notification won't be shown.
A few games with borderless screen, also won't be shown.

Reset

Reset All Settings

Cleanup all saved settings, and restart TwitchTalkApp as default.
Voice settings, Replacement Settings, Blocked List, Login State, all of them will erased. You can't restore them after reset.

TwitchTalkApp Version Release Note

ONLY IN JAPANESE. (sorry)

Auto Update

If new version is available, this app will notify it when you launch.
When you press "OK" button, autoupdate will start. Please use latest version as much as possible.
If you failed autoupdate, please download manually and overwrite it.
Saved password and settings won't be erased by overwriting.
