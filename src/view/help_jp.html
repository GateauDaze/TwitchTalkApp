基本の使い方

  1. 棒読みちゃんを起動しておきます。
  2. Connect with Twitchをクリックして、Twitchにログインします。
  3. Channel欄に読み上げたいチャンネルIDを入力します。
    ※自分のチャンネルであれば自分のチャンネルID、他人のチャンネルであれば相手のチャンネルIDです
    ※カンマ区切りで複数チャンネルを同時受信可能です。例：channel1,channel2,channel3
  4. Connectボタンを押すと読み上げが開始されます。
  5. Disconnectボタンを押すと読み上げが終了されます。
    ※PCをスリープする際などはDisconnectしてからスリープしてください。再接続できなくなります。

動作ログの出し方（デバッグ用）

  1. TwitchTalkApp.exeの起動オプションに「-log」を指定すると動作ログが出力されます。
  2. 起動オプションの指定は、コマンドプロンプトから実行するか、ショートカットの作成が簡単です。

Settings

Replacement

コメント内の特定文字列を、別に言葉に読み替えます。
和文と英文、それぞれに別の置換設定が適用されます。
英単語を日本語に読み替えると、英語専用音声では再生されなくなります。
Emoticonsでは、Twitchデフォルトエモートの読み替えを設定できます。

※エモートの読み替えは、和文・英文の読み替えの前に行われます。
Pronunciation欄に読み上げ方を入力し、Applyボタンで適用されます。
読み替え後の単語が和文ならば棒読みちゃんで、英文ならば英語音声で読み上げます。
和文・英文で置換設定された単語に置き換えた場合、読み替えがされます。

JP Voice Settings

棒読みちゃんとの接続設定ができます。
棒読みちゃんのSocket通信ポートが50001ではない場合に変更ができます。
棒読みちゃんを別PCで実行している場合、ホストアドレスを変更できます。

EN Voice Settings

英語音声の音量、再生速度、音程を変更できます。
Use English Voiceのチェックを外すと、英文も棒読みちゃんで読み上げされます。
PCの環境によっては変更が反映されない場合もあります。

Edit Blocked Users

コメント読み上げを行わないユーザーを指定できます。
初期段階では、BOTアカウントであるNightbotが設定されています。
コメントが投稿されると、該当ユーザーが自動でリストアップされます。
Status列のトグルスイッチをUnRead側に入れると読み上げされなくなります。

Options

Read Username

チェックが付いている時、コメント投稿者のユーザー名を読み上げます。
ユーザー名も読み替えの対象です。

Read Emoticons

チェックが付いている時、コメント本文に含まれるデフォルトエモートを読み上げます。
デフォルトエモート以外のエモートは常時読み上げられます。

Use English Voice

チェックが付いている時、英語音声をSAPI5で読み上げます。
未チェック時は日本語と同じ設定で英文が読み上げられます。

Use Notification

チェックが付いている時、コメントをポップアップ通知します。
フルスクリーンゲームをプレー中はポップアップは表示されません。
一部のボーダレスフルスクリーンでも表示されないことがあります。

Reset

Reset All Settings

保存されている設定を全て消去し、初期化状態でTwitchTalkAppsを再起動します。
音声設定、読み替え設定、ブロックリスト、ログイン情報などの全てが消去されます。
消去したデータは復元できません。

TwitchTalkApp Version Release Note

Auto Update

アプリ起動時、新しいバージョンが利用できる場合は通知されます。
OKボタンを押すと自動アップデートが開始されます。
自動アップデートに失敗した場合は、手動でダウンロードしてファイルを上書きしてください。
保存されたパスワードや設定などは消去されません。
